## 欢迎来到 **Wiki@LitLUG**

## 我们是谁

[**柳州铁树 Linux 用户协会「LUG@Lit」**](https://litlug.wiki)是由两名来自**柳州高级中学**和**柳州铁一中学**的学生自主创建的 **GNU/Linux** 爱好者团体，是一个以推广自由软件文化、宣传自由软件价值为目的的协会组织。

!!! tip "我们是一个独立运营的组织"

    我们不隶属于任何学校、团体，**Wiki@LitLUG** （下称**本 Wiki**）中所有内容均与 **柳州高级中学** 和 **柳州铁一中学** 无关。
